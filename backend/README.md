# Routes for Backend
## Routes

Route                      Method        Use

`/home`                      GET         Home Page

`/event/new`                 POST        Add a event

`/event/delete/:event_id`          POST        Delete a event

`/event/:event_id`                 GET         Get single event

`/user/:user_id`                  GET         Get User Profile

`/event/register/:event_id`        POST        Register User for event
