# Routes for Backend
## Routes

Route                      Method        Use

`/home`                      GET         Home Page

`/event/new`                 POST        Add a event

`/event/delete/:id`          POST        Delete a event

`/event/:id`                 GET         Get single push

`/user/:id`                  GET         Get User Profile

`/event/register/:id`        POST        Register User for event
